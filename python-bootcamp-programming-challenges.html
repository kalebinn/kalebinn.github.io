Introduction

Now that we've covered the very basics of input/output, conditional statements, loops, lists and functions. We can begin to do various programming challenges. Here is a collection of all the programming challenges that we will be covering today. These programming challenges are pulled from various places such as HackerRank, LeetCode, various textbooks and myself. I'm only including the material we have covered thus far. You will also be given an assignment since we will not be meeting until Monday. We will go over the programming challenges in the assignment together.

Challenge # 1

Let's do some warm up.
Define a function that prints out all the numbers between 0 and N2. Where N is an integer taken from stdin.

Challenge # 2

Still warm up.
Define a function that prints out all the numbers between 0 and N (inclusive). Where N is an integer taken from stdin. Put the numbers into a list and return this list.

Challenge # 3

Alright, here we go.
Define a function that takes a list as an input. return the maximum of the list and print the result.

Challenge # 4

Define a function that takes a list as an input. return the maximum even number in the list and print the result. If there are no even numbers return None.

Challenge # 4

Define a function that takes a list as an input. return the sum of all odd numbers in the list and print the result.

Challenge # 5

Define a function that forms a list of all perfect squares between 1 and N(inclusive), where N is a user input.

Challenge # 6

Implement the selection sort algorithm in ascending order.

Challenge # 7

- Inspired by HackerRank

Define a function that takes a list as an input. The function should return the maximum sum of any two elements.
